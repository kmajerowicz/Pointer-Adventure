1. **Create src/hooks/useFavorites.ts** — Hook that provides:
   - `favoriteIds: Set<string>` — from store
   - `favorites: Favorite[]` — from store
   - `toggleFavorite(routeId: string): Promise<void>` — Optimistic UI pattern:
     a. Read `isFav = favoriteIds.has(routeId)` before any mutation
     b. If isFav: call `removeFavoriteId(routeId)` + `removeFavorite(routeId)` immediately
     c. If !isFav: call `addFavoriteId(routeId)` immediately
     d. Try server call: if isFav → `supabase.from('favorites').delete().eq('user_id', user.id).eq('route_id', routeId)`; if !isFav → `supabase.from('favorites').insert({ user_id: user.id, route_id: routeId }).select().single()`
     e. On success of insert: call `addFavorite(data)` with the returned row
     f. On error: rollback — if was isFav: `addFavoriteId(routeId)`; if was !isFav: `removeFavoriteId(routeId)` + `removeFavorite(routeId)`
   - `updateNote(routeId: string, note: string | null): Promise<void>` — Calls `supabase.from('favorites').update({ note }).eq('user_id', user.id).eq('route_id', routeId)`. On success: `store.updateNote(routeId, note)`. On error: revert to previous note value.
   - `loadFavorites(): Promise<void>` — Fetches `supabase.from('favorites').select('*').eq('user_id', user.id)`, calls `setFavorites(data)`.
   - Import user from `useAuthStore` — `const user = useAuthStore((s) => s.profile)`. If `!user`, all mutations are no-ops (return early).
   - Use `useEffect` to call `loadFavorites()` when user changes (user?.id as dep).

2. **Create src/hooks/useActivity.ts** — Hook that provides:
   - `walkedIds: Set<string>` — from store
   - `entries: ActivityLogEntry[]` — from store
   - `logWalk(routeId: string): Promise<void>` — Inserts `{ user_id: user.id, route_id: routeId, walked_at: new Date().toISOString() }` into `activity_log`. On success: `addWalkedId(routeId)`, `appendEntry(data)`, show toast "Zapisano spacer!" using existing toast pattern. On error: show toast "Nie udalo sie zapisac spaceru".
   - `loadActivity(): Promise<void>` — Fetches walked route IDs: `supabase.from('activity_log').select('route_id').eq('user_id', user.id)`. Calls `setWalkedIds(data.map(d => d.route_id))`.
   - `loadActivityHistory(): Promise<void>` — Fetches full entries with route info: `supabase.from('activity_log').select('id, walked_at, created_at, user_id, route_id, route:routes(id, name, length_km)').eq('user_id', user.id).order('walked_at', { ascending: false })`. Calls `setEntries(data)`.
   - Use `useEffect` to call `loadActivity()` when user changes.
   - Import user from `useAuthStore((s) => s.profile)`.

3. **Create src/hooks/useInvites.ts** — Hook that provides:

   - `loadInvites(): Promise<void>` — Fetches `supabase.from('invitations').select('*').eq('created_by', user.id).order('created_at', { ascending: false })`. Calls `setInvitations(data)`.
   - Use `useEffect` to call `loadInvites()` when user changes.
