<task type="auto">
  <name>Task 1: Extend ui.ts store and create useInstallPrompt hook</name>
  <files>src/stores/ui.ts, src/hooks/useInstallPrompt.ts</files>
  <action>
1. **Extend `src/stores/ui.ts`** — add two new state fields and their setters:
   - `trailViewCount: number` — initialized to 0, incremented when TrailDetail mounts
   - `installPromptDismissedAt: number | null` — initialized from `localStorage.getItem('installPromptDismissedAt')` parsed as number, or null
   - `incrementTrailViewCount: () => void` — increments trailViewCount by 1
   - `dismissInstallPrompt: () => void` — sets installPromptDismissedAt to `Date.now()` AND persists to `localStorage.setItem('installPromptDismissedAt', String(Date.now()))`

   Keep all existing fields and setters unchanged. The localStorage persistence for dismissInstallPrompt ensures the 7-day cooldown survives browser refresh.

2. **Create `src/hooks/useInstallPrompt.ts`:**
   - Captures `beforeinstallprompt` event in useState, prevents default to defer it
   - Detects iOS Safari: `/iphone|ipad|ipod/i.test(navigator.userAgent)`
   - Detects standalone mode: `window.matchMedia('(display-mode: standalone)').matches`
   - Reads `trailViewCount` and `installPromptDismissedAt` from `useUIStore`
   - Computes `shouldShow`: `trailViewCount >= 3 AND (deferredPrompt !== null OR isIOS) AND NOT isStandalone AND (installPromptDismissedAt is null OR Date.now() - installPromptDismissedAt > 7 * 24 * 60 * 60 * 1000)`
   - Exposes `promptInstall` async function: calls `deferredPrompt.prompt()`, awaits `userChoice`, sets deferredPrompt to null
   - Returns `{ shouldShow, isIOS, promptInstall, dismiss }` where dismiss calls `useUIStore.getState().dismissInstallPrompt()`

   Import the `BeforeInstallPromptEvent` type from the global declaration in `vite-env.d.ts` (it is globally declared, no import needed).
  </action>

  </verify>
  <done>ui.ts has trailViewCount and installPromptDismissedAt with localStorage persistence. useInstallPrompt hook manages deferred prompt, iOS detection, standalone detection, 7-day cooldown, and 3-trail-view threshold. Build succeeds.</done>
